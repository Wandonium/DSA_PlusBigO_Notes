- What is the Union Find => it's a data structure that keeps track of elements which are split into one or more disjoint sets. It has two primary operations: 
    - Find => tells you what group a given element belongs to. 
    - Union => merges two groups together.

- Where is Union Find used => 
    - Kruskal's minimum spanning tree algorithm
    - Grid percolation
    - Network Connectivity
    - Least Common Ancestor in trees
    - Image Processing
